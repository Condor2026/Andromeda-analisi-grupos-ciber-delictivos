# Política de Seguridad

## Reportar Vulnerabilidades

Si encuentras información sensible expuesta en este repositorio o crees que hay un error que pueda poner en riesgo a terceros, por favor:

1. **No abras un issue público**
2. Contacta a: **Condor2026**

## Lo que hacemos

- Anonimizamos todos los datos personales
- No publicamos credenciales completas
- Revisamos la información antes de publicar

## Versiones

Este es un proyecto de investigación, no un software. Las actualizaciones se realizan cuando hay nueva información disponible.
